# 📝 React Todo App

A modern Todo application built with React that helps users manage tasks efficiently. It supports task creation, filtering, and persistent storage using Local Storage.

---

## 🚀 Features

- ➕ Add new tasks
- 🗑️ Delete tasks
- ✔️ Mark tasks as completed
- ⏳ Filter tasks (All / Completed / Pending)
- ✔️ Mark all tasks as completed
- 💾 Persistent storage using Local Storage
- ⚡ Fast and responsive UI

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- CSS
- Local Storage API

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/todo-app.git
```

### 2. Navigate to project folder

```bash
cd todo-app
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run the app

```bash
npm start
```

App will run at:
```
http://localhost:3000
```

---

## 💡 How It Works

- React state manages tasks
- Local Storage saves data in browser
- Filters are handled using conditional rendering
- UI updates instantly on state change
